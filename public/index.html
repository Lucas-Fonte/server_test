Server Test

This is a test server that creates and login users, connected to mongoDB + JWT to validate sessions.


Github

- URL https://github.com/Lucas-Fonte/server_test

Instalation

  1. Clone the project
  2. Enter the project directory
  3. Run npm install or yarn install

Usage

  • Endpoints

    • Health

      Details

      - URL https://lucasfonte-server-test.herokuapp.com/health

      - Method: GET

      - Headers: x-api-key: <app_x_api_key>

      - Response: json { "health": true }

    • Users/SignIn

      Details

      - URL https://lucasfonte-server-test.herokuapp.com/users/signIn

      - Method: POST

      - Headers: x-api-key: <app_x_api_key>

      - Request: json { "email": <email_string> *[required]*, "password": <password_string> *[required]* }

      - Response: json { "user": { "id": <user_id>, "name": <user_name>, "email": <user_email>, }, "token": <session_token> }

    • Users/SignUp

      Details

      - URL https://lucasfonte-server-test.herokuapp.com/users/signUp

      - Method: POST

      - Headers: x-api-key: <app_x_api_key>

      - Request: json { "name": <name_string> *[required]*, "email": <email_string> *[required]*, "password": <password_string> *[required]*, "phones": [ { "phone_number": <phone_number_integer>, "ddd": <ddd_integer> } ] *[optional]* }

      - Response: json { "id": <user_id>, "name": <user_name>, "email": <user_email>, "token": <session_token> }

    • Users/Search

      Details

      - URL https://lucasfonte-server-test.herokuapp.com/users/{userId}

      - Method: GET

      - Headers: x-api-key: <app_x_api_key> authorization: Bearer <user_token>

      - Response: json { "user": { "id": <user_id>, "name": <user_name>, "email": <user_email>, }, }
